  • 低摩擦体验：无需注册、登录或打开网页，一条命令就能开始对话，对终端老手而言是“纯净”的交互方式。

  • Linux 偏好明显：虽声称跨平台，但终端环境在 Windows 上的体验（如 PowerShell 对 ANSI 支持不佳）远不如 Unix-like 系统，可能劝退大量 Windows 用户。

  • 社区活跃：维护者持续更新站点列表和反爬虫机制，相比很多“年更”开源项目，Maigret 的生命力更可靠。

  • 性能瓶颈：扫描 3000+ 站点需要大量并发请求，容易触发目标网站的反爬机制，且长时间运行会拖垮本地网络或 IP 被封，不适合高并发任务。
💬 Maigret 是 OSINT 领域的“信息收割机”，但劝你用它查前男友前先想清楚——要么进局子，要么被反噬。
