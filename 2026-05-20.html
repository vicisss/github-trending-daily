📖 项目简介

codegraph 是一个为 AI 代码助手（如 Claude Code、Codex、Cursor）预构建的代码知识图谱工具。它通过本地索引项目结构、类型定义和依赖关系，生成结构化的图数据，让 AI 在理解项目时能大幅减少 token 消耗和工具调用次数，从而提升响应速度和准确性，尤其在大型代码库中效果显著。

🎯 适合人群

  • 使用 AI 代码助手（如 Claude、Cursor 等）的开发者和团队
  • 需要加速大型代码库 AI 理解效率的后端/全栈工程师
  • 负责优化 AI 工具链成本的工程经理或技术决策者
  • 对代码知识图谱技术感兴趣的产品经理（尤其是开发工具类产品）
  • 正在构建本地化、隐私优先的 AI 开发工作流的团队

💡 思维发散

  • “AI 代码体检” SaaS 服务：为开发团队提供代码库的“知识图谱即服务”，自动分析依赖、API 变更影响，并以可视化面板展示。竞争对手如 Sourcegraph 的 Code Intelligence，但 codegraph 更轻量、本地化且专注 AI 优化。商业化潜力高（企业订阅模式）。
  • CI/CD 智能代码审查插件：将 codegraph 集成到 GitHub Actions 或 GitLab CI 中，每次 PR 自动生成图级变更摘要，让 AI 审查只关注受影响模块，而非全量代码。现有竞品如 CodeRabbit 缺乏图结构理解，codegraph 可降低 50%+ token 成本，差异化明显。
  • 与 RAG 结合打造“私有代码问答机器人”：codegraph 作为索引层，配合本地向量数据库（如 Chroma）和 LLM，让企业能安全地提问“这个函数被哪些服务调用了？”而无需上传代码到云端。商业价值强，可对标 Glean 但更轻量。
  • 代码重构风险预测工具：利用知识图谱的依赖关系，预测某个函数改动影响的范围，并给出重构建议。可与 SonarQube 静态分析互补，形成“图+规则”的双重检查。市场空白，但需验证准确率。
