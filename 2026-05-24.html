📖 项目简介

CodeGraph 是一个为 Claude Code、Codex、Cursor 等 AI 编码助手提供“预索引代码知识图谱”的工具。核心思路是：在本地将你的代码库分析并构建成结构化的知识图谱（如函数调用关系、类依赖等），然后在 AI 编码时直接引用这个图，而不是让 AI 每次都去反复解析整个代码库。这能大幅减少 token 消耗、降低工具调用次数，并让 AI 的上下文理解更精准。

🎯 适合人群

  • AI 编码工具的重度用户：日常依赖 Claude Code、Cursor 等工具编写或重构代码，对 token 成本和响应速度敏感。
  • 大型代码库的维护者：项目规模大、模块多，AI 难以快速理解全局依赖关系。
  • 技术决策者/架构师：想要为团队引入更高效的 AI 辅助开发流程，降低 AI 工具的使用成本。
  • 对知识图谱技术好奇的开发者：想了解如何将结构化知识（如代码关系）与 LLM 结合，提升推理效率。

💡 思维发散

  • “代码审计 SaaS 插件”：将 CodeGraph 包装成一个 VS Code / JetBrains 插件，提供“一键生成代码依赖图”功能，并向企业客户收取订阅费。竞品如 CodeSee 和 Sourcegraph 依赖云端或全量索引，而 CodeGraph 更轻量、本地化且对 AI 友好。商业化价值：中期，面向中大型开发团队。

  • “跨项目代码搜索与复用平台”：企业内多个项目共享 CodeGraph 索引，实现跨仓库的“语义搜索”：输入“支付模块的 validate 函数”，直接定位到所有相关实现。竞品如 GitHub Code Search 依赖文本匹配，而 CodeGraph 能理解调用链。商业化价值：高，适合大型企业，按用户量或仓库数收费。
